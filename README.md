# Student Information Management System

## Degree Program

Bachelor of Computer Science

## Group Number

Group 02

## Project Description

A web-based Student Information Management System for primary and secondary schools in Tanzania. The system allows user management, student registration, viewing student records and searching students by registration number.

## Project Overview

This project was developed as part of the Open Source Technologies (CP 222) course. The system helps school administrators manage student information efficiently. It provides functionalities for registering students, displaying student records, searching students by registration number, and managing system users.

## Installation Steps

### Required Dependencies

* XAMPP (Apache and MySQL)
* PHP
* MySQL Database
* Git
* GitHub Account
* Web Browser

### Installation Procedure

1. Install XAMPP.
2. Start Apache and MySQL services.
3. Create a database named `student_db`.
4. Import the SQL database file from the `database` folder.
5. Copy the project folder into the XAMPP `htdocs` directory.
6. Open a browser and visit:

   ```
   http://localhost/student_management
   ```
7. Register a user and begin using the system.
